# palworld

```
NAME:
   palworld-manager - Palworld REST API client for CLI

USAGE:
   palworld-manager <command> [options] [arguments]

COMMANDS:
   announce               Announce the message to logged-in players
   force-stop             Force stop the palworld server (not recommended)
   kick                   Kick the specified logged-in player
   print-config-template  Print config template
   save                   Save the palworld server data
   show-information       Show the palworld server information
   show-metrics           Show metrics of the palworld server
   show-players           Show list of the logged-in players
   show-settings          Show settings of the palworld server
   shutdown               Shutdown the palworld server
   version                Print command version
   help, h                Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help (default: false)
```
